# ArrowMod 🏹

## Описание
Мод для Minecraft Fabric позволяет стрелять **10 стрелами** с алмазным мечом одним кликом!

## Требования
- **Minecraft:** 1.20.1
- **Loader:** Fabric 26.1.2+
- **Java:** 17+

## Установка

1. Скачайте `arrowmod-1.0.0.jar` из [Releases](https://github.com/Chasegrifer/ArrowMod/releases)
2. Положите файл в папку `%appdata%\.minecraft\mods` (Windows) или `~/.minecraft/mods` (Mac/Linux)
3. Запустите Minecraft с Fabric

## Как использовать

1. **Нажмите Left Alt** - включить/отключить режим стрельбы
2. **Левая кнопка мыши** - выстрелить 10 стрелами
3. У вас должны быть:
   - ✅ Алмазный меч в инвентаре
   - ✅ Минимум 10 стрел

## Команды
- **Left Alt** - Включение/отключение режима
- **Left Click (ЛКМ)** - Выстрел 10 стрелами
